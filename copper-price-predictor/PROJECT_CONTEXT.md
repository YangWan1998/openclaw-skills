## 4. 关键决策记录

### 决策1：数据源选择
- **问题**：Yahoo Finance API 在中国大陆返回403
- **决策**：改用 akshare 获取上海期货交易所数据
- **结果**：稳定获取CU0主力连续合约数据

### 决策2：新闻搜索方案
- **问题**：初期使用模拟数据
- **决策**：接入真实网络搜索（search_copper_news.py）
- **结果**：每次搜索约40条新闻，AI批量情感分析

### 决策3：学习闭环策略
- **问题**：如何持续优化预测准确度
- **决策**：分两步实现
  - 第一步：记录+归因+建议（人工确认后应用）✅
  - 第二步：满足条件时自动调整（待启用）⏳
- **原因**：万洋大魔王要求保持控制，避免AI擅自改动

### 决策4：cron任务设计
- **任务1**：8:30 新闻搜索（预存数据）
- **任务2**：9:00 生成日报（使用预存新闻）
- **目的**：避免9:00时新闻搜索超时导致日报延迟
